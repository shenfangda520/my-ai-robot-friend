# 我的AI机器朋友 🤖

一个简单的AI聊天机器人网页应用，可以和AI朋友聊天、问问题、听笑话！

## 功能特点

- 💬 与AI朋友聊天解闷
- 💡 询问问题获取回答
- 🎯 请求讲笑话
- 📚 学习新知识
- 🎨 美观的现代化界面设计

## 如何使用

1. 直接打开 `index.html` 文件即可运行
2. 在输入框中输入你想说的话
3. 点击"发送"按钮或按回车键发送消息

## 技术栈

- HTML5
- CSS3 (渐变、动画、响应式设计)
- JavaScript (ES6+)

## 项目结构

```
我的ai机器朋友/
├── index.html      # 主页面
├── style.css       # 样式文件
├── script.js       # JavaScript逻辑
└── README.md       # 项目说明
```

## 特色功能

- 响应式设计，支持手机和电脑
- 美观的渐变背景和卡片式布局
- 平滑的动画效果
- 智能的AI回复系统
